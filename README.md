# Xadrez ♟️
Projeto desenvolvido em C com o objetivo de simular os movimentos das principais peças do xadrez.

# 📌 Descrição
Foi desenvolvido em 3 níveis, com foco no desenvolvimento do raciocínio lógico e no domínio de estruturas de repetição e recursividade.
- Estruturas de repetição (for, while, do-while)
- Loops aninhados
- Recursividade
- Controle de fluxo (break e continue)

# 🛠️ Funcionalidades
♜ Torre
- Recursividade

♝ Bispo
- Loops aninhados

♛ Rainha
- Recursividade

♞ Cavalo
- Loop avançado com múltiplas variáveis, continue e break
